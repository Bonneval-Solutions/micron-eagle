## Commands

```bash
npm run dev          # Start Next.js + Slice Machine UI concurrently
npm run build        # Production build
npm run lint         # ESLint
npm run format       # Prettier
npm run slicemachine # Slice Machine UI only (port 9999)
```

`npm run dev` runs two processes: Next.js dev server and the Slice Machine UI. Both are needed when doing content modeling work.

## Architecture

**Stack:** Next.js (App Router) + Prismic CMS + TypeScript + CSS Modules. Deployed to Vercel.

### Prismic Content Model

Two custom types:
- **`page`** (repeatable) — all website pages, routed by UID. Has a `slices` zone and SEO fields.
- **`layout`** (singleton) — global site config: header/footer logos, nav links, contact info, social links.

Route mapping in `src/prismicio.ts`:
- UID `home` → `/`
- All other UIDs → `/:uid`

### Data Flow

1. `src/prismicio.ts` — creates the Prismic client (force-cache in prod, 5s revalidation in dev)
2. `src/app/layout.tsx` — fetches the `layout` singleton and passes it to `<Header>` and `<Footer>`
3. `src/app/page.tsx` and `src/app/[uid]/page.tsx` — fetch pages by UID, pass `page.data.slices` to `<SliceZone>`
4. `<SliceZone>` maps each slice to a component via `src/slices/index.ts` (all dynamic imports)

### Slices

Each slice lives in `src/slices/<SliceName>/` with:
- `index.tsx` — the React component (`SliceComponentProps<Content.XxxSlice>`)
- `index.module.css` — scoped styles
- `model.json` — Prismic content model definition

Access slice fields via `slice.primary` (single fields) and `slice.items` (repeatable group).

Use Prismic React helpers for rendering: `PrismicRichText`, `PrismicNextImage`, `PrismicNextLink`.

### Generated Types

`prismicio-types.d.ts` is auto-generated by Slice Machine — never edit it manually. Run `npm run slicemachine` and push model changes to regenerate.

### Previews & Simulator

- Draft previews: `/api/preview` (enter) and `/api/exit-preview` (exit)
- Slice Simulator: `http://localhost:3000/slice-simulator` — for visually testing slices in isolation
